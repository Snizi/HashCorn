# HashCorn

Tool designed to crack hashes using the rainbow tables tecnic

### Prerequisites
- Python 3.x

### Installing
- Download and install Python 3.x
- Clone the repo

### Modules
* crypt
* sys
* os



## Running the script
- python3 hashcorn.py wordlist.txt
